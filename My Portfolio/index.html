Birthday's List

I created a simple and interactive React app that displays a list of names and their respective birthdays. The application utilizes React’s state management with the `useState` hook to manage the list of names. The app dynamically updates the displayed count of birthdays and renders the names in a list format. I implemented a "Clear" button that, when clicked, empties the list of names, showcasing basic interaction and state updates. The layout is minimalistic and designed to provide a smooth user experience. This app also highlights the use of React's `map` function for rendering dynamic content and conditional rendering for updating the UI.
